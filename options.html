Channel Age Watchdog

Paste your personal YouTube Data API v3 key. It is stored only in this browser and is sent only to Google's API — never anywhere else.

No key yet? Get one free: open the Google Cloud Console, enable YouTube Data API v3, then create credentials under Credentials. When asked what data you'll access, choose Public data — this gives a plain API key (no OAuth sign-in needed, since we only read public channel info). The channels.list calls this extension makes cost 1 unit each (10,000/day free quota). Tip: restrict the key to the YouTube Data API v3.
